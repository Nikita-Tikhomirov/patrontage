# Patronage payment demo

Локальный пакет для тестовой страницы оплаты Точка Банка на MODX-сайте.

## Что развернуто

- Публичная демо-страница: https://patronage-service.ru/oplata
- MODX resource id: `1476`
- Страница сейчас не создает реальную платежную ссылку: нужен `merchantId` терминала с чеками и подтвержденные права JWT на интернет-эквайринг.
- JWT не хранится в репозитории и не вшит в HTML.

## Локальная сборка

```bash
node tools/payment-page.mjs
```

Команда генерирует `dist/oplata.html`.

## Проверки

```bash
node --test tests/payment-page.test.mjs
```

Проверяется нормализация суммы, клиентская валидация и наличие способов оплаты картой и СБП.

## Что нужно для следующего шага

От клиента/Точки нужно получить:

- `merchantId` терминала, который выбивает чеки;
- подтверждение, что JWT имеет доступы к интернет-эквайрингу;
- параметры фискализации: название услуги, НДС, система налогообложения;
- разрешение на тестовую оплату 1 рубль.
